# 🧮 Calculadora em C#

Projeto de calculadora desenvolvido em **C#**, com o objetivo de praticar lógica de programação, estrutura de projetos e versionamento utilizando Git e GitHub.

---

## 🚀 Tecnologias utilizadas

* C#
* Git
* GitHub

---

## 📂 Estrutura do projeto

```
Calculadora/
│
├── Calculadora/
│   ├── Program.cs
│   ├── Calculadora.csproj
│
├── Calculadora.sln
├── README.md
├── .gitignore
├── .gitattributes
```

---

## 🎯 Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Interface via console
* Entrada de dados pelo usuário

---

## 📚 Objetivo do projeto

Este projeto foi desenvolvido com foco em aprendizado e prática dos seguintes conceitos:

* Estrutura de projetos em C#
* Organização de código
* Lógica de programação
* Uso do Git e GitHub
* Versionamento de código
* Boas práticas de desenvolvimento

---

## 📌 Projeto para portfólio

Projeto criado para estudo e evolução na área de **desenvolvimento back-end com C# e .NET**, fazendo parte da construção de portfólio profissional.
